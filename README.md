# :information_source: How To Use

1- Clone the repository

```bash
https://github.com/williamjayjay/rentx-cars-mobile
```
2- Install the dependencies
```bash
yarn
```

3- Run json server api
```bash
yarn api
```

4- Run the application in device android
```bash
yarn android
```
